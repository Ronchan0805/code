• git add xxx.txt
  • git commit -m "message"
  • git status
  • git log(提交日志)
  • git reflog(命令日志)
  • git reset --hard 091fex....(版本回退)
  • git checkout -- read.html (丢弃工作区的修改)
  • git reset HEAD read.html (丢弃暂存区的修改)(需要配合git checkout -- read.html使用)
  • git push origin main (提交工作区内容到远程)
  • git pull(从远程库拉取更新内容)
